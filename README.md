# FilmoRatio

This is a simple symfony movies/tv series micro-api project.

## Installation

### Requirements

- PHP 8
- Composer
- Symfony
- MySQL

### Install

- Clone the project
- Install the vendors
- Create the database

#### Clone the project

git clone https://github.com/hhhallan/nfs__symfony-controle.git

#### Install the vendors

cd nfs__symfony-controle
composer install

#### Create the database

doctrine:database:create
